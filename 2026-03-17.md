        fun filterList(list:List<Int>, condition : (Int)-> Boolean ): List<Int>{
            
            val result=arrayListOf<Int>()
            for(item in list){
                if(condition(item)){
                    result.add(item)
                }
            }
            return result
        }

**Run**


Same as with ,but invokes as an extension function

Can be used to limit the scope of variables

run is a scope function that:

Executes a block of code on an object


Uses this as the reference


Returns the last expression


**Syntax:**

          object.run {
          // use this
          }

          
        run {
        // no object, just a code block
        }


**Example:**

          fun main(){
          val result = "Kotlin".run {
              println(this)
              length
          }
          
          println(result) // 6
          
          }

**Apply**

apply is a scope function used to:

Configure or initialize an object


Use this inside the block


Return the same object



**Syntax:**

        object.apply {
            // configure object using this
        }
  ------------------------------------------------------------------------------------------------------------

            fun main(){
            val person = Person().apply {
                name = "John"
                age = 25
            }
            
            }
            

No need to repeat person.

Returns the configured object


**Without apply vs With apply**

**Without:**

          val person = Person()
          person.name = "John"
          person.age = 25

          
**With:**
          val person = Person().apply {
             name = "John"
             age = 25
          }
          Cleaner and more readable

**also**

also is used to:

Perform side effects (logging, debugging, extra actions)

Use it as the object reference

Return the same object

**Syntax**

          
          object.also {
              // use it
          }


          
**Example**

          fun main(){
          val name = "Kotlin".also {
              println("Value is $it")
          }
          
          println(name) // Kotlin
          }


**Debugging in Chains**

          fun main(){
          val result = listOf(1, 2, 3)
              .map { it * 2 }
              .also {
                  println("After map: $it")
              }
              .filter { it > 2 }
          
          println(result)
          }

Perfect for checking intermediate results

**Logging While Creating Object**

        fun main(){
        val user = User("John", 25).also {
            println("User created: $it")
        }
        }

Does not affect object creation

**Validation / Side Checks**

          fun main(){
          val number = 10.also {
              require(it > 0) { "Number must be positive" }
          }
          }

Useful for validations without breaking flow
