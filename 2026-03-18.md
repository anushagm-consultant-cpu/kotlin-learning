**Extensions:**


Extensions allow you to write functions/properties as if they belong to a class, even if you don’t own that class.


Without the need to inherit from that class.


Can extend:

	Functions
  
	Properties
  
	Companion object


**Extension Functions:**


Can add functions to classes that we don’t own or can’t modify


Can access the object using the ‘this’ reference

The original classes is not modified

The new functions are not actually inserted in the class

Rather,its a shortcut to make the functions available using the usual ‘object.function’ notation

**Syntax:**

        fun ClassName.functionName(): ReturnType {
            // use 'this' to refer to the object
        }



**Example:**

            
            fun main() {
               val text="Hello Kotlin"
                println(text.reverseWords())
              
               
            }
            fun String.reverseWords():String{
                return this.split(" ").reversed().joinToString(" ")
            }

**Lazy:**

Lazy variables are variables that are initialized when needed

Useful to save memory in case the variable is never accessed

lazy is a delegate that initializes a value only when it is first used (not before).


**Syntax**

        
        val value: Type by lazy {
           // initialization logic
        }

**Example**


              val message: String by lazy {
                 println("Initializing...")
                 "Hello Kotlin"
              }
              
              fun main() {
                 println("Start")
                 println(message) // initialized here
                 println(message) // reused, not initialized again
              }
Output:

        Start
        
        Initializing...
        
        Hello Kotlin
        
        Hello Kotlin



**Rules of lazy**

1. Only for val (immutable)
var name by lazy { "Kotlin" } //  NOT allowed

2. Cannot reassign
val name by lazy { "Hello" }

name = "New" //  error



        import kotlin.random.Random
        fun main() {
            
            val someLargeVariable:String by lazy{
                "some large value"
            }
            if(Random.nextInt() % 2==0){
                println(someLargeVariable)
            }
        }
        


| Feature          | `lazy`       | `lateinit` |
| ---------------- | ------------ | ---------- |
| Type             | `val` only   | `var` only |
| Initialization   | Automatic    | Manual     |
| When initialized | First access | You decide |
| Null safety      | ✅ Safe       | ❌ Risky    |
| Reassign         | ❌ No         | ✅ Yes      |
