**Object**

An object in Kotlin is an instance of a class that contains properties and methods defined by the class.




                    class Person {
                        var name = ""
                        var age = 0
                    
                        fun introduce() {
                            println("My name is $name and I am $age years old")
                        }
                    }
                    
                    fun main() {
                        val person1 = Person()   // object
                        person1.name = "Alice"
                        person1.age = 22
                    
                        person1.introduce()
                    }




**Constructors**

**Primary Constructor**
The primary constructor is part of the class header. it goes after the class name. It is the most concise way to declare and initialize properties.



    // 'val' or 'var' in the header automatically creates properties
    class Student(val name: String, var grade: Int)
    
    fun main() {
        val s1 = Student("Rahul", 10) // Initializes name and grade immediately
    }

    

**The init Block**

The primary constructor cannot contain any code. If you need to perform logic during initialization (like validation), you use the init block.
        
        
        class User(val username: String) {
            init {
                println("User $username created.")
                require(username.isNotEmpty()) { "Username cannot be empty!" }
            }
        }

        
**Secondary Constructors**
A class can also have one or more secondary constructors, prefixed with the constructor keyword. These are useful if you need different ways to create an object.




                    class Student {
                        var name: String
                        var age: Int
                    
                        constructor(name: String, age: Int) {
                            this.name = name
                            this.age = age
                        }
                    }
                    
                    fun main() {
                        val s1 = Student("Bob", 21)
                        println(s1.name)
                    }




Rule: Every secondary constructor must delegate to the primary constructor using the this keyword.
        
        
        class Person(val name: String, val age: Int) {
            
            // Secondary constructor
            constructor(name: String) : this(name, 0) {
                println("Secondary constructor called for $name")
            }
        }
        
        fun main() {
            val p1 = Person("Amit", 25) // Calls Primary
            val p2 = Person("Sita")      // Calls Secondary, which calls Primary
        }
